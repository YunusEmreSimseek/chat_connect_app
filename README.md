# Chat Connect App

This application is an application where you can message and communicate with the people you want, and also stay in constant communication by sharing daily situations.

Stay connected to the conversation by creating an account in just a minute.

### Purpose

The only purpose of doing this application is to improve myself.

### Features

- Chats : The page where you can see the people you are in contact with and make new contacts.
- Home : The page where you can share all kinds of situations with the people you are in contact with on a daily basis and see what is shared.
- Settings : The page where you can access your profile page, change the language and log out of the account.
- Profile : The page where you can access and change the detailed information of your account.
